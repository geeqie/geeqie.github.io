Keyboard and Mouse Shortcuts

The tables below shows the defaults for keyboard shortcuts. The current settings for shortcuts may be found in the Keyboard Options dialogue.

Only the main window shortcuts can be redefined in Keyboard Options . The shortcuts in other windows are fixed.

In the list below, KP means KeyPad
  • Keyboard shortcuts in all windows, where relevant
  • Main window programmable keyboard shortcuts
  • Collections window non-programmable keyboard shortcuts
  • Duplicates window non-programmable keyboard shortcuts
  • Image window non-programmable keyboard shortcuts
  • Pan window non-programmable keyboard shortcuts
  • Search window non-programmable keyboard shortcuts
  • Mouse actions
  • Arrow keys actions
