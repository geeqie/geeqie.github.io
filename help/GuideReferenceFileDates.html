File date types

Geeqie understands 4 types of file dates:

  • Standard date (mtime) - The date the file contents were last modified, or when the file was created if since unchanged.

  • Status changed (ctime) - The date the file status was last changed. This includes change of owner, change of read/write permission and change of contents.

  • Exif date original (Exif.Photo.DateTimeOriginal) - For images from a digital camera, this is the time the photo was taken.

  • Exif date digitized (Exif.Photo.DateTimeDigitized) - For scanned images, this the time the scan was made. For images from a digital camera, it may be identical to the above date.
